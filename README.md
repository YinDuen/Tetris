# 俄羅斯方塊 Tetris

使用 HTML、CSS、JavaScript 製作的經典俄羅斯方塊遊戲，無需安裝，在瀏覽器中即可遊玩。

## 如何執行

1. 用瀏覽器直接開啟 `index.html`，或
2. 使用本地伺服器（例如 VS Code Live Server、`npx serve`）開啟專案資料夾後訪問首頁。

## 部署到 GitHub Pages

專案已設定 GitHub Actions：每次 push 到 `main` 或 `master` 會自動部署到 GitHub Pages。

1. 將此專案 push 到你的 GitHub 倉庫。
2. 在倉庫 **Settings → Pages** 中，將 **Source** 設為 **GitHub Actions**。
3. 之後每次 commit 並 push， workflow 會自動執行，完成後遊戲會出現在 `https://<你的帳號>.github.io/<倉庫名>/`。

## 操作方式

| 按鍵 | 功能 |
|------|------|
| ← → | 左右移動方塊 |
| ↑ | 旋轉方塊 |
| ↓ | 軟降（加速下落） |
| 空白鍵 | 硬降（一次落底） |
| P | 暫停 / 繼續 |

遊戲開始：按 **空白鍵** 或點擊「開始遊戲」按鈕。

**手機遊玩**：在手機或平板上開啟網頁後，畫面下方會出現觸控按鈕（← → 旋轉 ↓ 落底、暫停），可直接點擊操作，無需鍵盤。

## 規則簡介

- 方塊會從頂部落下，用方向鍵移動與旋轉，讓方塊填滿一整行即可消除該行。
- 消除行數會轉換為分數，等級會隨消除行數提升，下落速度會變快。
- 當新方塊無法出現在頂部時遊戲結束，可點「再玩一次」重新開始。

## 音效與音樂

- **背景音樂**：依**關卡**切換不同 BGM（關卡 1～5 各一首，關卡 6 起循環），每首為自創旋律 + 低音，暫停時靜音、結束後停止。
- **消除音效**：僅在消除行數時播放（依 1～4 行播放不同音高）。  
皆由 Web Audio API 產生，無需額外音檔。部分瀏覽器需先與頁面互動後才會播放。

## 檔案說明

- `index.html` — 頁面結構與畫布
- `style.css` — 介面與樣式
- `game.js` — 遊戲邏輯（方塊、移動、消行、計分、鍵盤控制、音效）
