# 俄羅斯方塊 Tetris

使用 HTML、CSS、JavaScript 製作的經典俄羅斯方塊遊戲，無需安裝，在瀏覽器中即可遊玩。

## 如何執行

1. 用瀏覽器直接開啟 `index.html`，或
2. 使用本地伺服器（例如 VS Code Live Server、`npx serve`）開啟專案資料夾後訪問首頁。

## 操作方式

| 按鍵 | 功能 |
|------|------|
| ← → | 左右移動方塊 |
| ↑ | 旋轉方塊 |
| ↓ | 軟降（加速下落） |
| 空白鍵 | 硬降（一次落底） |
| P | 暫停 / 繼續 |

遊戲開始：按 **空白鍵** 或點擊「開始遊戲」按鈕。

## 規則簡介

- 方塊會從頂部落下，用方向鍵移動與旋轉，讓方塊填滿一整行即可消除該行。
- 消除行數會轉換為分數，等級會隨消除行數提升，下落速度會變快。
- 當新方塊無法出現在頂部時遊戲結束，可點「再玩一次」重新開始。

## 音效

僅在**消除行數**時播放音效（依消除 1～4 行播放不同音高），使用 Web Audio API 產生，無需額外音檔。部分瀏覽器需先與頁面互動後才會播放。

## 檔案說明

- `index.html` — 頁面結構與畫布
- `style.css` — 介面與樣式
- `game.js` — 遊戲邏輯（方塊、移動、消行、計分、鍵盤控制、音效）
